## 使用说明

### 启动应用
```bash
# 应用启动后自动进入新版仪表盘
# 路径: PermissionWrapper → DashboardPage
```

### 主要操作

1. **主题切换**
   - 点击顶部栏的主题图标
   - 支持：浅色 / 深色 / 跟随系统

2. **模拟数据**
   - 点击播放按钮启动实时数据
   - 速度会自动在 0→40→0 km/h 循环
   - GPS 和模块状态会随机变化

3. **速度单位**
   - 点击速度下方的 KMH/MPH 按钮
   - 实时转换：km/h × 0.621371 = mph

4. **横竖屏**
   - 旋转设备自动切换布局
   - 竖屏：上下分屏
   - 横屏：左右分栏

5. **信息区域**
   - 点击卡片标题展开/折叠
   - 包含：控制器、BMS、行程、拓展模块
